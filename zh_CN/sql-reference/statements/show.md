
## SHOW DATABASES
查看数据库中所有数据库。
```SQL
show databases
```

## SHOW TABLES
查看某个 `database` 下所有的table。
```SQL
show tables
```

## SHOW CREATE DATABASE
获取指定 database 的 SCHEMA
```SQL
show create database database_name
```

## SHOW CREATE TABLE
获取指定 table 的 SCHEMA
```SQL
show create table table_name
```

## SHOW LICENCES
获取系统的 Licence 信息
```SQL
SHOW LICENCES  
```
## SHOW CLUSTER
获取集群节点信息。注：该指令仅在集群模式下生效。  
```SQL
SHOW CLUSTER
```
