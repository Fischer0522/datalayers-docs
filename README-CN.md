## 预览

请先确保已经安装 `python3`，且安装 `PyYAML` 模块：

```
pip install PyYAML
```

启动预览：

```sh
./preview.sh 8080
```

现在浏览器打开 <http://localhost:8080/zh/>, 如果更新了 dir.yaml 文件, 你需要重新运行上面的命令。
